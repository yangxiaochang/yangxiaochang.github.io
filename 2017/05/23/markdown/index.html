Markdown 语法入门

Tips: 符号与标题间要有空格！！！

...... 总共六级标题

列表

无序列表

在文字前加-或*

有序列表

在文字前直接加1.或2.或3.

Tips: 同样符号与文字间要有空格！！！

插入图片或链接

插入图片和插入链接的格式仅相差一个！字符

插入图片

格式如下：![替代图片的文字](图片URL)

Tips: URL的获取方式有多种
